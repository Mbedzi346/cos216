HTTP GET METHOD

This is the GET http method used to requests data from a specified resource/server. In this screen-shot this method is used to retrieve API json array from Jsonplaceholder API. The content returned is that of a user profile

HTTP POST METHOD

This is the POST http method that can be used to send data to the resource/server, for example, customer information, file upload, etc. In this screen-shot the post method sent data to create a new post to the Jsonplaceholder API. The feedback status returned was ok (200), meaning it was approved. The data returned is the new created post plus the Id of that post.
